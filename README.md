# BC--LM2596--ADJ
DC-DC switching regulator based on LM2596-ADJ  Adjustable output voltage using feedback resistor network  Designed and simulated using KiCad  High efficiency compared to linear regulators  Optimized PCB layout for low ripple and stability  Supports wide input voltage range (as per datasheet)  Includes schematic, PCB layout, Gerber files, BOM
